How to use

  1. Report Text — highlight from "Kentucky Uniform Police Traffic Collision Report" down to "School Bus Related   03 – Not Applicable", then paste into the Report Text box.
  2. Coordinates — highlight from the first "DEG:" label down through the last digit of the bottom "MIN" value, then paste into the Coordinates box.
  3. Intersection — type the intersection name or pick a previously used one from the dropdown.
  4. Page # — type the page number for this report.
  5. Click Parse Input — a row appears in the output table. Unit 1–4 and Notes columns are left blank for manual entry after pasting.
  6. Click Copy & Clear — copies all rows as tab-separated values (ready to paste into Excel) and clears the form and table.
