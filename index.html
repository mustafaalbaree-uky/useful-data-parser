How to use

  1. Report Text — in the PDF, highlight from "Kentucky Uniform Police Traffic Collision Report" down to "School Bus Related   03 – Not Applicable", then paste into the Report Text box.
  2. Coordinates — highlight from the first "DEG:" label down through the last digit of the bottom "MIN" value, then paste into the Coordinates box.
  3. Page # — type the page number for this report.
  4. Is Intersection? — choose Y, N, ?, or N/A from the dropdown.
  5. Has Diagram? — check the box if the report includes a diagram.
  6. Click Parse Input — a row appears in the output table below.
  7. Click Copy & Clear — the row is copied to your clipboard as tab-separated values, ready to paste into Excel. The form and table clear for the next report.
