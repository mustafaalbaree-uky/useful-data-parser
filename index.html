How to use

  1. New Project — click New Project, select a PDF file, give the project a name, and click Process PDF. All entries are extracted automatically.
  2. Table view — shows all extracted entries. Rows highlighted in blue have unverified fields. Click any row to open the Verification view for that entry.
  3. Verification view — review each highlighted field. Edit the value if needed, then click Verify to confirm it. Use ◄ ► to move between entries.
  4. View in PDF — in the Verification view, click this button and re-select your PDF file to render the source page as an image for reference.
  5. Export — use Copy Table (TSV) to copy all rows for pasting into a spreadsheet, or Download CSV to save a file.
  6. Manual Entry — click + Manual Entry in the table view to add a single entry by pasting report text and coordinates.
