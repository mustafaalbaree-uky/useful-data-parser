How to use

  1. New Project — click New Project, select one or more PDF files, give the project a name, and click Process PDF. All entries are extracted automatically.
  2. Add PDF — open a project and click + Add PDF in the topbar to append entries from additional PDF files.
  3. Table view — shows all extracted entries. Blue-highlighted cells have unverified fields. Click any row to open the Verification view for that entry.
  4. Verification view — review each highlighted field. Edit the value if needed, then click Verify to confirm it. Use ◄ ► to move between entries.
  5. View in PDF — click this button to render all pages for the current report. The PDF is cached for the session after the first pick.
  6. Export — use Copy Table (TSV) to copy all rows for pasting into a spreadsheet, or Download CSV to save a file.
  7. Classic Mode — available from the main menu for manual single-entry parsing without the project system.
